 # ERIGE (Ethans.Retro.Isometric.Game.Engine)

 ### Decription

This is gonna be a retro top down rpg engine written in C++ from scratch. Ill will be using this for learning how to build a 3D game engine using a modern graphics API. I will be following the kohi game engine series for this up to the renderer, then implemnting my own rednerer code to get familiar with vulkan. 

 ### Goals
 This engine is going to be small, not a full blown 3d engine that can render crazy models with super realistic lighting. The end goal is to make a game from this engine and for both to be used at portfolio peices.
